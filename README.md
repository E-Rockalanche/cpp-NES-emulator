# C++ NES Emulator (name pending)

A cycle acurate NES emulator written in c++

## Controls
configurable through config.json

### Port 1
* A: X
* B: Z
* Start: enter
* Select: right shift
* Directional buttons: arrow keys

### Port 2
* Zapper trigger: left mouse button

## Mappers working
0. NROM
1. MMC1
2. UxROM
3. CNROM
4. MMC3
