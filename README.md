# C++ NES Emulator (name pending)

A catch-up style cycle acurate NES emulator written in c++

## Controls

### Port 1
* A: V
* B: C
* Start: enter
* Select: spacebar
* Directional buttons: arrow keys

### Port 2
* Zapper trigger: left mouse button

## Mappers working
0. NROM
1. MMC1
2. UxROM
3. CNROM
4. MMC3
